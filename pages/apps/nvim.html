Neovim Setup Guide

Installing folke/tokyonight.nvim Neovim Theme

These instructions assume that you have Neovim (version 0.5 or later) installed, along with a plugin manager of your choice. The setup assumes that the plugin manager is already configured in your Neovim setup. If not, you'll need to install the respective plugin manager first. The instructions provided below include sane defaults for setting up the tokyonight.nvim theme and applying the colorscheme.

Popular Neovim Plugin Managers

vim-plug

Plug 'folke/tokyonight.nvim'
colorscheme tokyonight

packer.nvim

use {
  'folke/tokyonight.nvim',
  config = function()
    require('tokyonight').setup({})
    vim.cmd('colorscheme tokyonight')
  end
}

lazy.nvim

{
  "folke/tokyonight.nvim",
  lazy = false,
  priority = 1000,
  opts = {},
  config = function()
    vim.cmd('colorscheme tokyonight')
  end
}

Less Popular Neovim/Vim Plugin Managers

Install using Git

If you are a Git user, you can install the theme and keep up to date by cloning the repository:

git clone https://github.com/folke/tokyonight.nvim ~/.config/nvim/plugged/tokyonight.nvim

If you're using one of the less common plugin managers, here are the respective instructions:

dein.vim

call dein#add('folke/tokyonight.nvim')
colorscheme tokyonight

vundle


Plugin 'folke/tokyonight.nvim'
colorscheme tokyonight

minpac

call minpac#add('folke/tokyonight.nvim')
colorscheme tokyonight

Pathogen (manual installation)

git clone https://github.com/folke/tokyonight.nvim ~/.vim/bundle/tokyonight.nvim

In your vimrc:

colorscheme tokyonight

Note: It's recommended to use one of the more modern plugin managers (like vim-plug, packer.nvim, or lazy.nvim) for a better experience and more features.
